# 音频编辑使用指南

## 1. 工作区结构

先在素材工作区选择音乐、录音或视频，再打开“音频编辑”。编辑器分成两个稳定区域：

- 左侧：按“编辑、修复与音色、合成与输出”分类的工具栏。
- 右侧：当前工具自己的完整工作台；切换工具时会替换整体操作界面。

工具切换不会清空当前素材、播放位置或选区。所有处理都保留原文件，新文件默认保存到原素材旁的 `Echovault编辑输出`；若素材来自手机接收任务，结果还会登记到对应的待回传目录。

## 2. 波形、播放与选区

需要时间定位的工作台会通过随程序打包的 FFmpeg 读取 WAV、FLAC、MP3、M4A、OGG、Opus 以及视频中的音轨，并绘制实际振幅波形。

媒体探测和波形解码在后台线程执行，不阻塞按钮和页面切换。程序按文件路径、大小和修改时间缓存最近八个素材；重复进入未变化的文件不会再次运行 FFmpeg。各工具只在首次打开时接收波形，隐藏页面不会随播放位置和选区更新。时间线还会缓存刻度与波形静态图层，因此播放时只重绘红色游标，拖动时只重绘蓝色选区。

| 操作 | 作用 |
| --- | --- |
| 单击波形 | 移动红色播放游标 |
| 在波形上拖动 | 建立蓝色时间选区 |
| 修改开始/结束输入框 | 以 0.001 秒精度调整选区边界 |
| 全选 / 清除选区 | 选择整段或回到整文件处理 |
| 鼠标滚轮 / 放大 / 缩小 | 缩放时间轴 |
| 适合选区 | 将当前选区铺满编辑区 |
| 显示全部 | 回到完整时间轴 |

编辑区只保留一个“播放/暂停”按钮：有选区时只试听选区，没有选区时试听当前素材。处理完成后，“最新结果”卡片提供独立的结果试听和打开位置入口，不与素材播放按钮重复。

## 3. 工具与处理范围

| 分类 | 工具 | 主要用途 |
| --- | --- | --- |
| 编辑 | 裁剪与淡化 | 按选区裁剪，并可设置首尾淡化 |
| 编辑 | 分段导出 | 按固定秒数连续导出多个片段 |
| 编辑 | 增益 | 按分贝放大或衰减 |
| 修复与音色 | 底噪抑制 | 使用 FFT 滤镜削弱稳定背景噪声 |
| 修复与音色 | 响度标准化 | 设置目标 LUFS 与真峰值 |
| 修复与音色 | 八段均衡器 | 调整 60 Hz 至 16 kHz 八段增益和左右平衡 |
| 修复与音色 | 变速与变调 | 独立设置速度与半音偏移 |
| 合成与输出 | 顺序拼接 | 按列表顺序首尾连接多个文件 |
| 合成与输出 | 多轨混合 | 叠加多个文件并设置各轨音量 |
| 合成与输出 | 提取音轨 | 从视频或音频容器导出独立音轨 |
| 合成与输出 | 音频标签 | 修改标题、歌手、专辑、年份和轨道号 |

各工具不再共用同一个中间处理区：裁剪提供精确起止时间、提取/删除片段、增益、速度、变调、延迟和淡化；降噪并排显示原始与处理后波形；均衡器使用八个纵向频段推子；拼接与混音使用带静音、独奏、音量和迷你波形的轨道区，并可把前两条轨道合成为左右声道；标准化、提取和标签分别使用目标卡、媒体音轨卡和音乐信息表单。

## 4. 标准操作流程

1. 选择素材并打开“音频编辑”。
2. 在左侧选择工具，进入对应的独立工作台。
3. 按工作台逻辑建立选区、轨道、频段或输出目标。
4. 修改参数和输出文件名。
5. 点击底部蓝色生成按钮，等待状态栏显示完成。
6. 在该工作台底部试听结果或打开所在位置；需要继续处理时，重新选择生成的文件作为素材。

## 5. 手机回传与安全性

手机通过 LocalSend 传入的文件会归入独立传输任务。编辑结果会根据源文件关联到该任务，自动出现在“手机传输”的待回传列表中；发送成功的待回传副本会移入已发送缓存，`Echovault编辑输出` 中的正式结果不会被删除。

所有处理先写入同目录临时文件，成功后再原子替换目标；同名文件会自动生成不冲突的新名称，源文件不会被覆盖。降噪适合持续、稳定的底噪，复杂人声串音或强烈环境声建议使用专门的分离或修复工具。
