# 音频编辑使用指南

## 1. 工作区结构

先在素材工作区选择音乐、录音或视频，再打开“音频编辑”。编辑器分成三个稳定区域：

- 左侧：按“编辑、修复与音色、合成与输出”分类的工具栏。
- 中间：真实波形、时间标尺、选区、播放和缩放控制。
- 右侧：当前工具的说明、处理范围、参数和执行按钮。

工具切换不会清空当前素材、播放位置或选区。所有处理都保留原文件，新文件默认保存到原素材旁的 `Echovault编辑输出`；若素材来自手机接收任务，结果还会登记到对应的待回传目录。

## 2. 波形、播放与选区

编辑区会通过随程序打包的 FFmpeg 读取 WAV、FLAC、MP3、M4A、OGG、Opus 以及视频中的音轨，并绘制实际振幅波形。

| 操作 | 作用 |
| --- | --- |
| 单击波形 | 移动红色播放游标 |
| 在波形上拖动 | 建立蓝色时间选区 |
| 修改开始/结束输入框 | 以 0.001 秒精度调整选区边界 |
| 全选 / 清除选区 | 选择整段或回到整文件处理 |
| 鼠标滚轮 / 放大 / 缩小 | 缩放时间轴 |
| 适合选区 | 将当前选区铺满编辑区 |
| 显示全部 | 回到完整时间轴 |
| 底部滚动条 | 在放大的时间轴中横向移动 |

编辑区只保留一个“播放/暂停”按钮：有选区时只试听选区，没有选区时试听当前素材。处理完成后，“最新结果”卡片提供独立的结果试听和打开位置入口，不与素材播放按钮重复。

## 3. 工具与处理范围

| 分类 | 工具 | 主要用途 |
| --- | --- | --- |
| 编辑 | 裁剪与淡化 | 按选区裁剪，并可设置首尾淡化 |
| 编辑 | 分段导出 | 按固定秒数连续导出多个片段 |
| 编辑 | 增益 | 按分贝放大或衰减 |
| 修复与音色 | 底噪抑制 | 使用 FFT 滤镜削弱稳定背景噪声 |
| 修复与音色 | 响度标准化 | 设置目标 LUFS 与真峰值 |
| 修复与音色 | 三段均衡器 | 调整低频、中频和高频增益 |
| 修复与音色 | 变速与变调 | 独立设置速度与半音偏移 |
| 合成与输出 | 顺序拼接 | 按列表顺序首尾连接多个文件 |
| 合成与输出 | 多轨混合 | 叠加多个文件并设置各轨音量 |
| 合成与输出 | 提取音轨 | 从视频或音频容器导出独立音轨 |
| 合成与输出 | 音频标签 | 修改标题、歌手、专辑、年份和轨道号 |

“裁剪与淡化”必须先建立选区。增益、底噪抑制、响度标准化、三段均衡器、变速与变调、分段导出会优先处理当前选区；没有选区时处理整个文件。选区处理会把选中的时间范围输出为一个新文件。拼接、混合、提取和标签按各自的文件列表或容器逻辑执行，不使用时间选区。

## 4. 标准操作流程

1. 选择素材并打开“音频编辑”。
2. 在中间波形上试听、定位并拖出需要处理的时间范围。
3. 在左侧选择工具，确认右侧显示的“当前素材”和“处理范围”。
4. 修改参数和输出文件名。
5. 点击右侧蓝色执行按钮，等待状态栏显示完成。
6. 在“最新结果”卡片试听结果或打开所在位置；需要继续处理时，重新选择生成的文件作为素材。

## 5. 手机回传与安全性

手机通过 LocalSend 传入的文件会归入独立传输任务。编辑结果会根据源文件关联到该任务，自动出现在“手机传输”的待回传列表中；发送成功的待回传副本会移入已发送缓存，`Echovault编辑输出` 中的正式结果不会被删除。

所有处理先写入同目录临时文件，成功后再原子替换目标；同名文件会自动生成不冲突的新名称，源文件不会被覆盖。降噪适合持续、稳定的底噪，复杂人声串音或强烈环境声建议使用专门的分离或修复工具。
