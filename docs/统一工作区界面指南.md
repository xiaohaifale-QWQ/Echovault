# 统一工作区界面指南

## 四个主入口

新版主窗口不再把所有功能放成平级标签，只保留四个任务入口：

1. **素材**：添加文件夹、切换音乐/视频、选择当前文件。
2. **歌词与标签**：识别、编辑、翻译、在线歌词、封面。
3. **音频编辑**：常用音频处理和人声分离。
4. **导出与传输**：手机接收、差异核对、回传和批量任务。

选择素材后，顶部素材卡会显示三个直接入口：“识别或匹配歌词”“封面与标签”“编辑音频”。
用户不需要先理解软件内部模块划分。

## AI 右侧抽屉

AI 助手默认隐藏，不占用正常工作空间。点击顶栏“AI 模式 → 启动”后，AI 从窗口右侧展开；
关闭 AI 后，中间工作区恢复完整宽度。原有在线/本地 AI、语音输入和命令确认规则不变。

## 音频编辑

音频编辑采用持续三栏：

- 左侧：基础编辑、音质处理、多文件处理、文件与输出。
- 中间：当前素材、波形、播放位置和最近结果。
- 右侧：当前工具输入、输出、详细参数和执行按钮。

切换工具只改变右侧参数，不会清空当前素材。处理仍然生成新文件，禁止覆盖输入原文件。

## 既有细节保留

本次重构只调整信息架构，不取消此前确认的规则：

- 手机接收目录的“打开目录”和“选择目录”保持分离。
- 在线歌词和封面使用同一结果区；点击封面后直接进入确认。
- 人声分离模型继续在统一模型库管理。
- 待回传、已发送缓存和缓存统计保持原有语义。
- 讯飞极速接口未授权时继续自动回退流式听写。
